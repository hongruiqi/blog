字符串类型和初始化

Swift中的字符串类型为String，其中包含若干Character类型的字符。

let greet = "hello world" // String
let letter = "a" // Character

字符串的常用操作

// 字符串初始化
var emptyString = ""
var anotherEmptyString = String()

// 字符串判空
emptyString.isEmpty

// 字符串连接
var a = "hello"
a += "world"
a = a + "!"
